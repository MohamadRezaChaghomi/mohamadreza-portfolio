# نت‌اپ | اپلیکیشن مدیریت یادداشت حرفه‌ای

## 📋 نمای کلی

نت‌اپ یک اپلیکیشن مدیریت یادداشت پیشرفته و حرفه‌ای است که در ابتدا به عنوان یک پروژه دانشگاهی مهندسی نرم‌افزار توسعه یافت. این پروژه که به عنوان یک سیستم مدیریت یادداشت وب ساده آغاز شد، به طور گسترده گسترش یافت و به یک اپلیکیشن جامع و آماده تولید با ویژگی‌های سطح سازمانی تبدیل شد.

این پروژه توسط محمدرضا چاقمی به عنوان توسعه‌دهنده فول‌استک انفرادی توسعه یافت و تمام الزامات را پیاده‌سازی کرده و به طور قابل توجهی فراتر از مشخصات اصلی با عملکرد پیشرفته، معماری مدرن و تجربه کاربری حرفه‌ای عمل کرد.

## ✨ ویژگی‌های کلیدی

### ✅ الزامات اصلی (مشخصات پروژه)
- **ثبت‌نام و احراز هویت کاربران**: سیستم کامل ثبت‌نام و ورود
- **امنیت CAPTCHA**: CAPTCHA یکپارچه برای امنیت صفحه ورود
- **ادغام OAuth**: پشتیبانی از احراز هویت حساب گوگل
- **عملیات CRUD یادداشت**: ایجاد، خواندن، به‌روزرسانی، حذف یادداشت با برچسب زمانی
- **جستجوی پیشرفته**: جستجوی متن کامل در هر دو زبان فارسی و انگلیسی

### 🚀 ویژگی‌های توسعه‌یافته (فراتر از الزامات)

#### 📝 مدیریت پیشرفته یادداشت
- **ویرایشگر متن غنی**: ویرایشگر کامل WYSIWYG با نوار ابزار قالب‌بندی
- **پوشه‌ها و سازماندهی**: سیستم پوشه سلسله‌مراتبی با کدگذاری رنگی
- **سیستم برچسب‌گذاری**: دسته‌بندی چند برچسبی با تخصیص رنگ
- **سطح اولویت**: سیستم چهار سطحی اولویت (کم، متوسط، بالا، فوری)
- **ستاره و پین**: علامت‌گذاری یادداشت‌های مهم برای دسترسی سریع
- **بایگانی و زباله‌دان**: مدیریت کامل چرخه حیات یادداشت‌ها
- **تاریخچه نسخه‌ها**: ردیابی خودکار تمام تغییرات
- **عملیات انبوه**: انتخاب چندگانه و پردازش دسته‌ای
- **تاریخ‌های سررسید و یادآورها**: مدیریت یادداشت‌های حساس به زمان
- **قابلیت‌های خروجی**: خروجی یادداشت‌ها در فرمت‌های متعدد

#### 📊 تحلیل‌های پیشرفته
- **آمار实时**: داشبورد زنده با معیارهای فعالیت کاربر
- **نمودارهای بصری**: نمودارها و تجسم داده‌های تعاملی
- **تحلیل کاربر**: ردیابی عملکرد فردی و جمعی
- **الگوهای استفاده**: بینش در روند ایجاد و ویرایش یادداشت
- **تحلیل ذخیره‌سازی**: نظارت بر استفاده از دیسک و بهینه‌سازی

#### 🎨 تجربه کاربری برتر
- **تم تاریک/روشن**: پشتیبانی کامل تم با انتقال‌های نرم
- **رابط کاربری مدرن**: رابط حرفه‌ای با ناوبری بصری
- **به‌روزرسانی‌های实时**: به‌روزرسانی زنده بدون تازه‌سازی صفحه
- **میانبرهای صفحه کلید**: میانبرهای افزایش بهره‌وری
- **کشیدن و رها کردن**: مدیریت بصری فایل و پوشه
- **دسترسی‌پذیری**: طراحی سازگار با WCAG
- **پشتیبانی آفلاین**: عملکرد جزئی بدون اینترنت

#### 🔒 امنیت سازمانی
- **احراز هویت پیشرفته**: نشست‌های امن مبتنی بر JWT
- **محدودیت نرخ**: محافظت در برابر حملات brute force
- **اعتبارسنجی ورودی**: اعتبارسنجی جامع سمت سرور
- **محافظت XSS**: پیشگیری داخلی از cross-site scripting
- **توکن‌های CSRF**: محافظت در برابر cross-site request forgery
- **کوکی‌های امن**: سیاست‌های HTTPOnly و Secure cookie
- **لاگ‌های حسابرسی**: ردیابی کامل تمام فعالیت‌های سیستم

## 🚀 شروع کار

### پیش‌نیازها
- Node.js 18+ و npm/yarn/pnpm
- پایگاه داده MongoDB (محلی یا Atlas)
- اعتبارنامه‌های Google OAuth (برای ورود اجتماعی)
- کلیدهای Google reCAPTCHA
- اعتبارنامه‌های سرویس ایمیل

### نصب

1. کلون کردن مخزن
```bash
git clone https://github.com/MohamadRezaChaghomi/note-app.git
cd note-app
```

2. نصب وابستگی‌ها
```bash
npm install
```

3. تنظیم محیط
فایل `.env.local` ایجاد کنید:

```env
# Database
MONGODB_URI=your_mongodb_uri

# Authentication
NEXTAUTH_URL=http://localhost:3000
NEXTAUTH_SECRET=your_secret_key

# Google OAuth
GOOGLE_CLIENT_ID=your_google_client_id
GOOGLE_CLIENT_SECRET=your_google_client_secret

# reCAPTCHA
NEXT_PUBLIC_RECAPTCHA_SITE_KEY=your_site_key
RECAPTCHA_SECRET_KEY=your_secret_key

# Email
EMAIL_SERVER=smtp.gmail.com
EMAIL_PORT=587
EMAIL_USER=your_email@gmail.com
EMAIL_PASSWORD=your_app_password
```

4. اجرای سرور توسعه
```bash
npm run dev
```

5. باز کردن در مرورگر
به [http://localhost:3000](http://localhost:3000) بروید
