# Pastry Supplies | پنل ادمین قنادی

## 📋 نمای کلی

پنل ادمین حرفه‌ای برای مدیریت قنادی با دسته‌بندی چندسطحی محصولات. این پروژه یک پنل ادمین کامل برای سایت قنادی است که با React پیاده‌سازی شده و امکان مدیریت محتوای استاتیک، کاربران و محصولات را فراهم می‌کند.

این پروژه توسط محمدرضا چاقمی به عنوان توسعه‌دهنده اصلی پنل ادمین در یک تیم توسعه یافت و مسئولیت پیاده‌سازی پنل ادمین و منطق frontend را بر عهده داشت.

## ✨ ویژگی‌های کلیدی

### ✅ الزامات اصلی (مشخصات پروژه)
- **مدیریت محتوای استاتیک**: ویرایش صفحات درباره ما و اطلاعات تماس
- **مدیریت کاربران (CRUD)**: عملیات کامل ایجاد، خواندن، به‌روزرسانی، حذف کاربران
- **دسته‌بندی چندسطحی محصولات**: ساختار درختی دسته‌بندی‌ها
- **مدیریت محصولات**: افزودن محصولات با تصاویر و توضیحات
- **گزارش‌گیری**: آمار کاربران و محصولات

### 🚀 ویژگی‌های توسعه‌یافته

#### 📝 مدیریت محتوا
- **ویرایشگر WYSIWYG**: ویرایش متن غنی برای صفحات استاتیک
- **مدیریت منو**: ساختار ناوبری پویا
- **SEO Optimization**: متا تگ‌ها و بهینه‌سازی موتور جستجو
- **مدیریت رسانه**: آپلود و مدیریت تصاویر

#### 👥 مدیریت کاربران پیشرفته
- **نقش‌ها و دسترسی‌ها**: سیستم نقش‌محور (ادمین، مدیر، کاربر)
- **پروفایل کاربران**: اطلاعات کامل و تاریخچه فعالیت
- **اعلان‌ها**: سیستم اعلان‌های ایمیل و داخل برنامه
- **آمار کاربران**: تحلیل رفتار و فعالیت‌ها

#### 🏷️ دسته‌بندی هوشمند
- **ساختار درختی**: دسته‌بندی‌های تو در تو تا ۵ سطح
- **ویژگی‌های پویا**: فیلدهای سفارشی برای هر دسته
- **SEO-friendly URLs**: آدرس‌های بهینه برای موتور جستجو
- **مدیریت انبوه**: عملیات دسته‌ای روی دسته‌بندی‌ها

#### 🛍️ مدیریت محصولات پیشرفته
- **ویژگی‌های پویا**: فیلدهای سفارشی بر اساس دسته
- **مدیریت موجودی**: ردیابی موجودی و هشدار کم بودن
- **قیمت‌گذاری هوشمند**: قیمت‌های چندگانه و تخفیف‌ها
- **تصاویر متعدد**: گالری تصاویر با مرتب‌سازی

#### 📊 گزارش‌گیری و تحلیل
- **داشبورد مدیریتی**: آمار实时 فروش و کاربران
- **گزارش‌های مالی**: درآمد، هزینه‌ها، سود
- **تحلیل رفتار**: الگوهای خرید و جستجو
- **خروجی پیشرفته**: گزارش‌های PDF و Excel

## 🚀 شروع کار

### پیش‌نیازها
- Node.js 16+
- npm یا yarn
- Backend API (در صورت نیاز)

### نصب

1. کلون کردن مخزن
```bash
git clone https://github.com/MohamadRezaChaghomi/PastrySupplies.git
cd PastrySupplies
```

2. نصب وابستگی‌ها
```bash
npm install
```

3. تنظیم متغیرهای محیطی (در صورت نیاز)
```env
REACT_APP_API_URL=http://localhost:3001/api
REACT_APP_UPLOAD_URL=http://localhost:3001/uploads
```

4. اجرای برنامه
```bash
npm start
```

5. باز کردن در مرورگر
به [http://localhost:3000](http://localhost:3000) بروید
