# خانه‌ی پدری | سامانه مدیریت املاک

## 📋 نمای کلی

خانه‌ی پدری یک سامانه جامع مدیریت املاک است که با هدف دیجیتال‌سازی کامل فرآیندهای یک دفتر املاک طراحی و توسعه یافته. این پروژه با ارائه امکانات پیشرفته‌ای مثل نمایش روی نقشه، سیستم تقویم، خروجی Excel و سیستم لاگ، یک محصول آماده برای استفاده واقعی است.

این پروژه توسط محمدرضا چاقمی به عنوان توسعه‌دهنده فول‌استک انفرادی توسعه یافت و تمام الزامات را پیاده‌سازی کرده و به طور قابل توجهی فراتر از مشخصات اصلی با عملکرد پیشرفته، معماری مدرن و تجربه کاربری حرفه‌ای عمل کرد.

## ✨ ویژگی‌های کلیدی

### ✅ الزامات اصلی (مشخصات پروژه)
- **مدیریت ملک‌ها**: افزودن و ویرایش اطلاعات ملک با تصاویر
- **نمایش روی نقشه**: ادغام نقشه برای نمایش موقعیت ملک‌ها
- **خروجی Excel**: استخراج گزارش‌ها در فرمت Excel
- **سیستم تقویم**: برنامه‌ریزی قرار ملاقات‌ها
- **لاگ امنیتی**: ثبت تمام عملیات سیستم

### 🚀 ویژگی‌های توسعه‌یافته (فراتر از الزامات)

#### 🏠 مدیریت پیشرفته ملک
- **تصاویر متعدد**: آپلود و مدیریت چندین تصویر برای هر ملک
- **دسته‌بندی هوشمند**: گروه‌بندی ملک‌ها بر اساس نوع (فروش، اجاره، رهن)
- **اطلاعات کامل**: متراژ، تعداد اتاق، امکانات، موقعیت جغرافیایی
- **جستجوی پیشرفته**: فیلتر بر اساس قیمت، متراژ، منطقه
- **وضعیت ملک**: موجود، رزرو شده، فروخته شده

#### 🗺️ نمایش نقشه پیشرفته
- **ادغام Leaflet**: نمایش تعاملی نقشه با نشان‌گذاری ملک‌ها
- **جستجوی مکانی**: یافتن ملک‌ها بر اساس موقعیت
- **نمای خیابان**: نمایش تصاویر خیابانی (در صورت موجود بودن)
- **فاصله‌ها**: محاسبه فاصله تا نقاط مهم (مدرسه، مترو، etc.)

#### 📊 گزارش‌گیری و خروجی
- **خروجی Excel پیشرفته**: گزارش‌های فیلتر شده با فرمت مناسب
- **آمار فروش**: نمودارهای فروش ماهانه و سالانه
- **گزارش مالی**: درآمد، هزینه‌ها، سود خالص
- **گزارش مشتری**: آمار مشتریان و ترجیحات

#### 📅 سیستم تقویم هوشمند
- **برنامه‌ریزی قرارها**: تنظیم وقت ملاقات با مشتریان
- **یادآور خودکار**: اعلان‌های ایمیل و SMS
- **تقویم تعاملی**: نمایش بصری قرارها و رویدادها
- **همگام‌سازی**: اتصال به Google Calendar

#### 🔒 سیستم لاگ و امنیت
- **لاگ کامل عملیات**: ثبت تمام تغییرات و دسترسی‌ها
- **ردیابی حسابرسی**: پیگیری تغییرات توسط کاربران
- **پشتیبان‌گیری خودکار**: نسخه‌های پشتیبان منظم
- **کنترل دسترسی**: سطوح مختلف دسترسی برای کاربران

#### 🎨 تجربه کاربری برتر
- **تم تاریک/روشن**: پشتیبانی کامل تم با انتقال‌های نرم
- **رابط کاربری مدرن**: طراحی حرفه‌ای و کاربرپسند
- **پاسخگویی کامل**: سازگاری با تمام دستگاه‌ها
- **دسترسی‌پذیری**: طراحی WCAG compliant

## 🛠️ پشته فناوری

### Frontend
- **Next.js 16**: فریمورک React با App Router
- **React 19**: React جدید با ویژگی‌های پیشرفته
- **Tailwind CSS**: فریمورک CSS utility-first
- **Leaflet**: کتابخانه نقشه تعاملی
- **React Query**: مدیریت وضعیت سرور
- **Framer Motion**: انیمیشن‌ها و انتقال‌ها

### Backend
- **Next.js API Routes**: APIهای serverless
- **MongoDB**: پایگاه داده NoSQL
- **Mongoose**: مدل‌سازی داده
- **JWT**: احراز هویت امن

### ابزارها و کتابخانه‌ها
- **ExcelJS**: تولید فایل‌های Excel
- **next-themes**: مدیریت تم
- **Lucide React**: آیکون‌ها
- **date-fns**: کار با تاریخ‌ها

### توسعه و استقرار
- **TypeScript**: توسعه type-safe
- **ESLint**: کیفیت کد
- **Vercel**: میزبانی تولید
- **MongoDB Atlas**: پایگاه داده ابری

## 🚀 شروع کار

### پیش‌نیازها
- Node.js 18+
- MongoDB (محلی یا Atlas)
- npm یا yarn

### نصب

1. کلون کردن مخزن
```bash
git clone https://github.com/MohamadRezaChaghomi/real-estate.git
cd real-estate
```

2. نصب وابستگی‌ها
```bash
npm install
```

3. تنظیم متغیرهای محیطی
فایل `.env.local` ایجاد کنید:

```env
MONGODB_URI=your_mongodb_connection_string
NEXTAUTH_SECRET=your_secret_key
NEXTAUTH_URL=http://localhost:3000
```

4. اجرای سرور توسعه
```bash
npm run dev
```

5. باز کردن در مرورگر
به [http://localhost:3000](http://localhost:3000) بروید
