# سامانه رزرواسیون | Reservation System

## 📋 نمای کلی

یک سامانه رزرواسیون عمومی با قابلیت استفاده برای هر کسب‌وکاری. این پروژه به صورت تیمی توسعه یافته و شامل پنل ادمین کامل، سیستم کد تخفیف، نظرات کاربران، تیکت‌دهی و اتصال به درگاه پرداخت واقعی است.

این پروژه در یک محیط تیمی توسعه یافت و من مسئولیت پیاده‌سازی پنل ادمین و صفحه جزئیات محصول را بر عهده داشتم.

## ✨ ویژگی‌های کلیدی

### ✅ الزامات اصلی (مشخصات پروژه)
- **پنل ادمین کامل**: مدیریت محصولات، سفارشات، تخفیف‌ها و نظرات
- **صفحه جزئیات محصول**: نمایش اطلاعات با امکان انتخاب تعداد
- **سیستم کد تخفیف**: کدهای تخفیف درصدی و مبلغی
- **مدیریت نظرات**: تایید و پاسخ به نظرات کاربران
- **اتصال درگاه پرداخت**: پرداخت واقعی با API بانکی

### 🚀 ویژگی‌های توسعه‌یافته

#### 👨‍💼 پنل ادمین پیشرفته
- **داشبورد مدیریتی**: آمار فروش، سفارشات و کاربران
- **مدیریت محصولات**: CRUD کامل با تصاویر و دسته‌بندی
- **مدیریت سفارشات**: پیگیری وضعیت سفارشات از ثبت تا تحویل
- **مدیریت تخفیف‌ها**: کدهای تخفیف با محدودیت استفاده
- **مدیریت کاربران**: پروفایل و تاریخچه خرید کاربران
- **گزارش‌گیری**: گزارش‌های مالی و آماری

#### 🛒 تجربه خرید کاربر
- **صفحه محصول**: نمایش جزئیات کامل با گالری تصاویر
- **سبد خرید**: مدیریت آیتم‌ها با امکان ویرایش
- **فرآیند پرداخت**: گام به گام با اعتبارسنجی
- **پیگیری سفارش**: وضعیت实时 سفارشات
- **تاریخچه خرید**: لیست سفارشات قبلی

#### 💰 سیستم پرداخت امن
- **درگاه پرداخت واقعی**: اتصال به API بانکی ایرانی
- **رمزگذاری داده‌ها**: امنیت اطلاعات کارت بانکی
- **تایید پرداخت**: مکانیزم‌های امنیتی چندلایه
- **رسید پرداخت**: تولید خودکار فاکتور
- **بازگشت وجه**: سیستم refund

#### 💬 سیستم نظرات و پشتیبانی
- **نظرات کاربران**: امتیازدهی و متن نظر
- **مدیریت نظرات**: تایید و پاسخ توسط ادمین
- **سیستم تیکت**: پشتیبانی مشتریان
- **چت زنده**: ارتباط实时 (اختیاری)

#### 🎯 سیستم بازاریابی
- **کدهای تخفیف**: درصدی، مبلغی، هدیه
- **کمپین‌های تبلیغاتی**: تخفیف‌های زمان‌دار
- **پاداش وفاداری**: امتیاز برای خریدهای مکرر
- **ارسال خبرنامه**: ایمیل مارکتینگ

## 🛠️ پشته فناوری

### Frontend
- **JavaScript (ES6+)**: زبان برنامه‌نویسی اصلی
- **HTML5**: ساختار معنادار و semantic
- **CSS3**: استایل‌دهی پیشرفته و responsive
- **DOM Manipulation**: کار با APIهای مدرن مرورگر

### Backend (تیم)
- **Node.js**: runtime JavaScript سمت سرور
- **Express.js**: فریمورک وب سریع
- **MongoDB**: پایگاه داده NoSQL
- **JWT**: احراز هویت امن

### ابزارها و کتابخانه‌ها
- **Payment Gateway API**: اتصال به درگاه‌های پرداخت ایرانی
- **Email Service**: ارسال ایمیل‌های تراکنشی
- **Image Upload**: مدیریت فایل‌های رسانه
- **Validation Libraries**: اعتبارسنجی داده‌ها

### توسعه و استقرار
- **Git**: کنترل نسخه
- **NPM**: مدیریت بسته‌ها
- **Development Server**: محیط توسعه محلی
- **Production Hosting**: سرورهای ابری

## 🚀 شروع کار

### پیش‌نیازها
- Node.js 14+
- MongoDB
- درگاه پرداخت (برای تست)

### نصب

1. کلون کردن مخزن
```bash
git clone https://github.com/MohamadRezaChaghomi/rezervation-pj.git
cd rezervation-pj
```

2. نصب وابستگی‌ها
```bash
npm install
```

3. تنظیم متغیرهای محیطی
```env
MONGODB_URI=mongodb://localhost:27017/reservation
JWT_SECRET=your_jwt_secret
PAYMENT_API_KEY=your_payment_key
EMAIL_SERVICE_KEY=your_email_key
```

4. اجرای سرور توسعه
```bash
npm run dev
```

5. باز کردن در مرورگر
- پنل ادمین: [http://localhost:3000/admin](http://localhost:3000/admin)
- فروشگاه: [http://localhost:3000](http://localhost:3000)
